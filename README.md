## Running Locally
### Using Docker
1. ```cd build/docker```
2. ```docker compose up -d```
3. Open ```http://localhost:31080``` or direct to the docs ```http://localhost:31080/api-docs```

### Using NPM
1. Execute ```load_env.sh``` to export env variables
2. ```npm run build```
